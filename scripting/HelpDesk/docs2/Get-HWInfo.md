# Get-HWInfo

## SYNOPSIS
Get hardware information.

## SYNTAX

```
Get-HWInfo [[-Computername] <String>] [<CommonParameters>]
```

## DESCRIPTION
Get server hardware detail

## EXAMPLES

### Example 1
```
PS C:\> get-hwinfo srv1

Name Version OS              FreeGB
---- ------- --              ------
SRV1 v2.0.0  Windows Unicorn   7.97
```
